Основные команды Linux

Изучите базовые команды для работы в терминале

Работа с файлами

Команда
Описание
Пример
ls
Показать содержимое директории
ls -la
cd
Перейти в директорию
cd /home/user
cp
Копировать файлы/директории
cp file.txt backup/
mv
Переместить или переименовать файл
mv old.txt new.txt
rm
Удалить файл/директорию
rm -r folder/
mkdir
Создать директорию
mkdir new_folder

Системные команды

Команда
Описание
Пример
sudo
Выполнить команду с правами root
sudo apt update
top
Показать запущенные процессы
top
ps
Показать текущие процессы
ps aux
kill
Завершить процесс
kill -9 1234
df
Показать свободное место на дисках
df -h
free
Показать использование памяти
free -m

Попробуйте в интерактивном терминале

user@linuxguide:~$
