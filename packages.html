Управление пакетами в Linux

Изучите различные системы управления пакетами в дистрибутивах Linux

APT (Advanced Package Tool)

Используется в дистрибутивах на основе Debian (Ubuntu, Linux Mint, Debian и др.).

Основные команды:

sudo apt update
Обновить список доступных пакетов
sudo apt upgrade
Обновить все установленные пакеты
sudo apt install <пакет>
Установить пакет
sudo apt remove <пакет>
Удалить пакет
sudo apt search <запрос>
Поиск пакета
sudo apt autoremove
Удалить ненужные зависимости

Дополнительные возможности:

  • Поддержка PPA (Personal Package Archives) для установки дополнительного ПО
  • Возможность отката обновлений
  • Широкий выбор пакетов в официальных репозиториях

YUM/DNF (Yellowdog Updater Modified/Dandified YUM)

Используется в дистрибутивах на основе Red Hat (Fedora, CentOS, RHEL). DNF - новая версия YUM.

Основные команды:

sudo yum update
Обновить все пакеты
sudo yum install <пакет>
Установить пакет
sudo yum remove <пакет>
Удалить пакет
sudo yum search <запрос>
Поиск пакета
sudo yum info <пакет>
Информация о пакете
sudo yum clean all
Очистить кеш

Особенности:

  • Автоматическое разрешение зависимостей
  • Поддержка групп пакетов
  • Плагины для расширения функциональности
  • DNF работает быстрее и эффективнее YUM

Pacman

Менеджер пакетов для Arch Linux и производных дистрибутивов (Manjaro, EndeavourOS и др.).

Основные команды:

sudo pacman -Syu
Обновить систему и пакеты
sudo pacman -S <пакет>
Установить пакет
sudo pacman -R <пакет>
Удалить пакет
sudo pacman -Ss <запрос>
Поиск пакета
sudo pacman -Qi <пакет>
Информация об установленном пакете
sudo pacman -Sc
Очистить кеш

Особенности:

  • Простота и скорость работы
  • Доступ к AUR (Arch User Repository) с тысячами пакетов
  • Минималистичный дизайн
  • Rolling release модель обновлений

Zypper

Менеджер пакетов для openSUSE и SUSE Linux Enterprise.

Основные команды:

sudo zypper refresh
Обновить список пакетов
sudo zypper update
Обновить все пакеты
sudo zypper install <пакет>
Установить пакет
sudo zypper remove <пакет>
Удалить пакет
sudo zypper search <запрос>
Поиск пакета
sudo zypper info <пакет>
Информация о пакете

Особенности:

  • Поддержка патчей и обновлений безопасности
  • Возможность отката изменений
  • Интеграция с YaST (система настройки openSUSE)
  • Поддержка нескольких репозиториев

Полезные советы по работе с пакетами

Добавление репозиториев

Многие дистрибутивы позволяют добавлять сторонние репозитории для доступа к дополнительному ПО. Например, в Ubuntu можно добавить PPA:

sudo add-apt-repository ppa:репозиторий/название
sudo apt update

Поиск зависимостей

Если пакет не устанавливается из-за зависимостей, можно попробовать:

sudo apt --fix-broken install # Для APT
sudo yum deplist <пакет> # Для YUM

Установка из исходников

Если пакета нет в репозиториях, можно установить его из исходного кода:

./configure
make
sudo make install

Сохранение списка пакетов

Перед переустановкой системы сохраните список установленных пакетов:

# Для APT
dpkg --get-selections > packages.list

# Для восстановления
sudo dpkg --set-selections < packages.list
sudo apt-get dselect-upgrade
